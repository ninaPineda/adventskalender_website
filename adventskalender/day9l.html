Tag 9

Hier sind die Lösungen:

  1. 😮🎄😮🎄 - Lösung: "O Tannenbaum"

    Die Emojis symbolisieren den Anfang des Liedes: "O Tannenbaum, O Tannenbaum".

    • Das "😮" steht für "O".
    • Das "🎄" steht für einen Tannenbaum.

  2. 🤫🌃😇🌃 - Lösung: "Stille Nacht"

    Die Emojis beschreiben das Lied "Stille Nacht".

    • Das "🤫" steht für "Stille".
    • Das "🌃" symbolisiert die Nacht.
    • Das "😇" deutet auf die "heilige Nacht".

  3. 🔙🎄 - Lösung: "Last Christmas"

    Die Emojis repräsentieren "Last Christmas".

    • Das "🔙" steht für "Last" (Vergangenheit).
    • Das "🎄" steht für Weihnachten (Christmas).

  4. 🔔🔔🔔 - Lösung: "Jingle Bells"

    Die Glocken ("🔔") weisen auf das Lied "Jingle Bells" hin.

  5. ♾️🎁🫵🏼 - Lösung: "All I Want for Christmas Is You"

    Diese Emojis beschreiben das Lied "All I Want for Christmas Is You".

    • Das "♾️" (Unendlichkeit) steht für "all".
    • Das "🎁" repräsentiert "Christmas".
    • Das "🫵🏼" bedeutet "you".

  6. 😮🫵🏼🤣😮🫵🏼😇 - Lösung: "O du fröhliche"

    Das Lied "O du fröhliche" wird hier beschrieben.

    • "😮" steht für "O".
    • "🫵🏼" bedeutet "du".
    • "🤣" (Lachen) steht für "fröhliche".

  7. ❄️☃️🌨️ - Lösung: "Let It Snow"

    Die Emojis stehen für das Lied "Let It Snow".

    • "❄️" symbolisiert Schnee.
    • "☃️" steht für winterliche Stimmung.
    • "🌨️" unterstreicht die Bedeutung von Schneefall.

  8. 🚗🏠🎄 - Lösung: "Driving Home for Christmas"

    Dieses Lied wird durch die Emojis beschrieben: "Driving Home for Christmas".

    • "🚗" bedeutet "Driving".
    • "🏠" steht für "Home".
    • "🎄" repräsentiert Weihnachten.

  9. 🤣🎄♾️ - Lösung: "Fröhliche Weihnacht überall"

    Das Lied "Fröhliche Weihnacht überall" wird hier dargestellt.

    • "🤣" steht für "Fröhliche".
    • "🎄" für "Weihnacht".
    • "♾️" symbolisiert "überall" (Unendlichkeit).

  10. ❄️⚪️👗 - Lösung: "Schneeflöckchen, Weißröckchen"

    Dieses Kinderlied wird durch die Emojis beschrieben:

    • "❄️" steht für Schneeflocken.
    • "⚪️" repräsentiert "Weiß".
    • "👗" steht für ein Röckchen.

  11. 🦌🔴🌟 - Lösung: "Rudolph the Red-Nosed Reindeer"

    Dieses bekannte Lied wird durch folgende Emojis dargestellt:

    • "🦌" steht für ein Rentier.
    • "🔴" symbolisiert die rote Nase von Rudolph.
    • "🌟" steht für die magische Weihnachtsstimmung.

  12. 🎶🎄🇪🇸 - Lösung: "Feliz Navidad"

    Die Emojis zeigen das Lied "Feliz Navidad".

    • "🎶" symbolisiert Musik.
    • "🎄" repräsentiert Weihnachten.
    • "🇪🇸" steht für den spanischen Text des Liedes.
